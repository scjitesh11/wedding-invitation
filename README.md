# 💍 Satish ❤️ Lipsa - Wedding Invitation

An elegant, interactive wedding invitation website built with HTML, CSS, and JavaScript.

## 🌟 Features

- **Beautiful Hero Section** - Stunning landing page with glass morphism effect
- **Wedding Invitation** - Formal invitation details
- **Live Countdown Timer** - Countdown to the wedding day
- **Event Schedule** - All wedding events (Haldi, Baraat, Ceremony, Reception)
- **Photo Gallery** - Interactive image gallery with lightbox
- **RSVP Form** - Collect responses via WhatsApp
- **Responsive Design** - Works on all devices
- **Elegant UI** - Traditional yet modern design with pink and white theme

## 📄 Pages

1. **index.html** - Home/Landing page
2. **invitation.html** - Wedding invitation details
3. **countdown.html** - Live countdown timer
4. **events.html** - Wedding events schedule
5. **gallery.html** - Photo gallery
6. **rsvp.html** - RSVP form
7. **thankyou.html** - Thank you message

## 🚀 Getting Started

1. Clone this repository
2. Add your images to the `images/` folder:
   - `couple.jpg` - Hero section background
   - `gallery1.jpg`, `gallery2.jpg`, `gallery3.jpg`, `gallery4.jpg` - Gallery images
3. Update the WhatsApp number in `script.js` (line with `wa.me/`)
4. Customize wedding details as needed
5. Deploy to GitHub Pages

## 📝 Customization

- Update couple names, dates, and venue in HTML files
- Change color scheme in `style.css`
- Update WhatsApp number for RSVP in `script.js`
- Add your own images

## 🎨 Design Features

- Glass morphism effect on hero section
- Gradient backgrounds
- Smooth hover animations
- Live countdown timer (updates every second)
- Image lightbox effect
- Mobile responsive layout

## 💬 RSVP Integration

RSVP responses are sent via WhatsApp to the configured number. Update the phone number in `script.js` to receive messages.

## 📱 Responsive

The website is fully responsive and works perfectly on:
- Desktop
- Tablet
- Mobile devices
